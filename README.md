# binary-calculator
1 week project for CS 495

## Installation
```git clone https://github.com/DrewThomasCorps/binary-calculator.git```

Set up JAVA_HOME and install Maven Dependencies.

To run the project select `Maven` -> `Plugins` -> `javafx` -> `javafx:run`
